h2. Introduction

p. Opinoid is a simple web application which syndicates news from a set of known RSS/Atom sources and provides visitors with a sorted list of articles (like Reddit), based on user's votes and comments. It uses "TornadoWeb":http://www.tornadoweb.org , "MongoDB":http://www.mongodb.org , "jQuery":http://jquery.com and "Bootstrap":http://twitter.github.com/bootstrap/ .

p. Requires python >= 2.6; if using python >= 2.7 you don't need to install the argparse module in your virtualenv.

h2. Get involved

p. Check for some deployment instructions in the "Deployment Guide":https://github.com/giulivo/opinoid/wiki/Deployment-Guide .

p. A live version is pushed online at "alpha.opinoid.com":http://alpha.opinoid.com (currently hosted on "OpenShift":http://openshift.redhat.com , backup url "opinoid-gfidente.rhcloud.com":http://opinoid-gfidente.rhcloud.com). Please report issues and ideas here on github!
